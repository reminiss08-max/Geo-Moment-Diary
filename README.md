# Geo-Moment-Diary

Aplikasi Android sederhana untuk mencatat momen berdasarkan lokasi menggunakan OpenStreetMap.

## Teknologi
- Kotlin
- Android Studio
- SQLite
- OSMDroid
- OpenStreetMap
